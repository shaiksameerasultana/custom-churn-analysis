# custom-churn-analysis
Customer Churn Analysis Dashboard
![Customer Churn Analysis Dashboard](WhatsApp-Image-2025-11-13

This project presents an interactive Power BI dashboard for Customer Churn Analysis, visualizing key insights from a telecom dataset. The dashboard helps stakeholders quickly identify important patterns in customer attrition, target segments, payment method trends, and churn by contract type, providing actionable intelligence to support data-driven retention strategies.

Features
Comprehensive Visualizations: Pie charts, bar charts, and summary metrics to break down churn by contract, gender, payment method, internet service, and monthly charges.

Interactive Analysis: Slicers for contract type, gender, and internet service allow users to filter and drill down into specific segments.

Key Metrics Highlighted: Displays overall churn numbers, monthly charges, tenure, and customer counts for rapid assessment.

Business Impact: Insights uncover which customer groups are at highest risk, helping teams focus retention efforts where they matter most.
